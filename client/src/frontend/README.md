# Frontend

Frontend React du projet ECB Production Manager.

## Lancer

Depuis la racine du projet :

```bash
npm install --prefix src/frontend
npm --prefix src/frontend start
```

Le frontend attend par defaut une API sur `http://localhost:8000/api`.

## Build

```bash
npm --prefix src/frontend run build
```

## Role actuel

Le frontend porte le shell applicatif Windows et les modules visibles du produit :

- Dashboard
- Import BOM
- Visualisation BOM
- Commande composant
- Machine PnP
- Parametre
