# Cahier Des Charges - Etat Courant

## Objectif produit

Fournir un outil desktop Windows pour preparer la production PCB a partir de BOM harmonisees.

## Flux prioritaires

1. importer une BOM texte type Eagle
2. harmoniser valeurs et footprints
3. relire et corriger la revision
4. stocker la revision harmonisee
5. rattacher des revisions a une production
6. generer une commande composants
7. preparer la vision machine PnP

## Cible

- priorite: desktop Windows
- support technique: frontend web local pour le developpement

## Exigences UI

- navigation laterale stable
- ecrans visibles tot
- pas de fausses donnees
- etats vides propres
- workflow metier lisible

## Hors scope actuel

- diffusion produit signee
- deploiement ops industriel
- internationalisation
- couverture frontend elevee sur toutes les pages
